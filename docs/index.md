## Descripción General

`pyotels` permite interactuar programáticamente con OtelMS para:
- Extraer información de reservas.
- Consultar disponibilidad de habitaciones.
- Automatizar reportes.

El proyecto está construido utilizando Python moderno, con soporte para operaciones asíncronas y validación de datos robusta.
